# Dogrulama Kontrol Listesi

- `npm install` temiz tamamlandi.
- `npm run db:migrate` SQLite semasini olusturdu.
- `npm run db:seed` demo verisini ekledi.
- `npm test` API testlerini gecirdi.
- `npm run build` API ve web buildlerini uretir.
- Login sonrasi dashboard gorunur.
- Satis kaydi tank stok seviyesini dusurur.
- Yetersiz stokta satis engellenir.
- Vardiya kapama acik vardiya uzerinde calisir.
- Entegrasyon testleri log kaydi uretir.
- Mali Operasyonlar sekmelerinde seed verisi gorunur.
- e-Fatura/e-Arsiv, e-Defter, EPDK, OKC ve beyanname dry-run islemleri durum gunceller.
- Banka mutabakati satis ve banka hareketlerini eslestirir.
- Genel muhasebe fisi borc/alacak dengesiyle uretilir.
